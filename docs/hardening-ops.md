# Hardening ops script

The `scripts/hardening-ops.sh` helper includes an `install_tailscale` function that configures Tailscale's official apt repository (keyring + sources list) and installs the `tailscale` package.

## References

- Tailscale install for Ubuntu (apt repo setup): https://tailscale.com/kb/1103/install-linux?tab=ubuntu
- Tailscale install for Debian (apt repo setup): https://tailscale.com/kb/1103/install-linux?tab=debian
